Un-Truncated_Results.csv gives output from an analysis of the read data with raw importance weights (i.e. normalized, but not truncated). Two runs were performed on each of 27 different parameter combinations. Listed variables include: index of parameter initialization, values of parameter initialization, parameter estimates, standard errors, correlation between estimates, obs data likelihood ratio from a fixed reference parameter value, and checks whether fitted standard errors are negative (suggesting numerical instability).
Similar story for Truncated_Results.csv.

Truncated_Estimate_Analysis.csv gives information about the chosen parameter estimate. Variables include estimate, SE and 95% Wald CI.
